# Explanation
The .text files here can be used to compile the LaTex paper.
I tried using python to call MitTex to compile the latex paper. But the process was buggy.
I used free version of https://www.overleaf.com/ instead. Just upload the .tex files to a new project, and compile.
Overleaf has a version hostory feature, but I found it more convenient to track history and review changes in .tex files in my git client.
