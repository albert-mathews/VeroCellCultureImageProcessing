# Explanation
The .tex files here can be used to compile the LaTex paper.
there are two 'main' .tex files. one formatted per PLOS bio journal LaTex template.
I used free version of https://www.overleaf.com/ Just upload the .tex files to a new project, and compile.
Overleaf has a version hostory feature, but I found it more convenient to track history and review changes in .tex files in my git client.
